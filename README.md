# 🚀 Developer Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and certifications as a Full Stack Web Developer.

## ✨ Features

- **Responsive Design** - Works seamlessly on all devices
- **Modern UI/UX** - Clean and professional interface with smooth animations
- **Interactive Navigation** - Smooth scrolling between sections
- **Certificate Showcase** - Display professional certifications with modal view
- **Experience Timeline** - Detailed education and work experience
- **Contact Form** - Easy way to get in touch
- **Section Scroll Button** - Quick navigation through all sections

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Google Fonts (Ubuntu, Text Me One)

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required!
